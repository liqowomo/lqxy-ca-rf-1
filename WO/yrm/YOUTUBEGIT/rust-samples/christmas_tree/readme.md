# Christmas Tree Exercise

## Learning Goal

* Practice working for `for` loops
* Practice working with slices of numeric values

## Requirements

Draw a christmas tree on the screen with `*` characters. Example

```txt
                *
              *****
            *********
              *****
            *********
          *************
        *****************
          *************
        *****************
      *********************
    *************************
  *****************************
*********************************
            *********
            *********
```
